# Metallurgy 4

Metallurgy 4 is a vast enhancement to the basic mining system, adding well over 50+ new metals . 
Our goal is to create a balanced atmosphere for players to explore and enjoy mining. Metallurgy 4 is also configurable, and entirely tailoring for the player's experience.
